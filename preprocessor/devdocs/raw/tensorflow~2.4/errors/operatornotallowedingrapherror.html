tf.errors.OperatorNotAllowedInGraphError

An error is raised for unsupported operator in Graph execution.

tf.errors.OperatorNotAllowedInGraphError(
    *args, **kwargs
)

For example, using a tf.Tensor as a Python bool in Graph execution is not allowed.
