numpy.flatiter.index

attribute

flatiter.index

Current flat index into the array.

Examples

>>> x = np.arange(6).reshape(2, 3)
>>> fl = x.flat
>>> fl.index
0
>>> next(fl)
0
>>> fl.index
1
