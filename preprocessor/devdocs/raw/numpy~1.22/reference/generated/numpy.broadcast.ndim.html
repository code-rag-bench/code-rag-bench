numpy.broadcast.ndim

attribute

broadcast.ndim

Number of dimensions of broadcasted result. Alias for nd.

New in version 1.12.0.

Examples

>>> x = np.array([1, 2, 3])
>>> y = np.array([[4], [5], [6]])
>>> b = np.broadcast(x, y)
>>> b.ndim
2
