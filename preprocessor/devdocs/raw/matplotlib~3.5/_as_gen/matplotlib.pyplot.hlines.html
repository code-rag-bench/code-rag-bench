matplotlib.pyplot.hlines

matplotlib.pyplot.hlines(y, xmin, xmax, colors=None, linestyles='solid', label='', *, data=None, **kwargs)[source]

Plot horizontal lines at each y from xmin to xmax.

Parameters
yfloat or array-like

y-indexes where to plot the lines.

xmin, xmaxfloat or array-like

Respective beginning and end of each line. If scalars are provided, all lines will have same length.

colorslist of colors, default: rcParams["lines.color"] (default: 'C0')
linestyles{'solid', 'dashed', 'dashdot', 'dotted'}, optional
labelstr, default: ''
Returns
LineCollection
Other Parameters
dataindexable object, optional

If given, the following parameters also accept a string s, which is interpreted as data[s] (unless this raises an exception):

y, xmin, xmax, colors

**kwargsLineCollection properties.
