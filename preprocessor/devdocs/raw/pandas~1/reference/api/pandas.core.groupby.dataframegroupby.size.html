pandas.core.groupby.DataFrameGroupBy.size

DataFrameGroupBy.size()[source]

Compute group sizes.

Returns
DataFrame or Series

Number of rows in each group as a Series if as_index is True or a DataFrame if as_index is False.

See also

Series.groupby
