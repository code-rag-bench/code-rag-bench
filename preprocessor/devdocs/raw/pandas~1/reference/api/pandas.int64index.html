pandas.Int64Index

classpandas.Int64Index(data=None, dtype=None, copy=False, name=None)[source]

Immutable sequence used for indexing and alignment. The basic object storing axis labels for all pandas objects. Int64Index is a special case of Index with purely integer labels. .

Deprecated since version 1.4.0: In pandas v2.0 Int64Index will be removed and NumericIndex used instead. Int64Index will remain fully functional for the duration of pandas 1.x.

Parameters
data:array-like (1-dimensional)
dtype:NumPy dtype (default: int64)
copy:bool

Make a copy of input ndarray.

name:object

Name to be stored in the index.
