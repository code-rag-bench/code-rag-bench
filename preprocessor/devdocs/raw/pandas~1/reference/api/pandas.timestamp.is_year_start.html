pandas.Timestamp.is_year_start

Timestamp.is_year_start

Return True if date is first day of the year.

Examples

>>> ts = pd.Timestamp(2020, 3, 14)
>>> ts.is_year_start
False

>>> ts = pd.Timestamp(2020, 1, 1)
>>> ts.is_year_start
True
