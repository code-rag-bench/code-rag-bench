# tf.raw_ops.StageSize

Op returns the number of elements in the underlying container.

        shared_name='', name=None

Args  
---  
`dtypes` |  A list of `tf.DTypes`.   
`capacity` |  An optional `int` that is `>= 0`. Defaults to `0`.

A `Tensor` of type `int32`.
