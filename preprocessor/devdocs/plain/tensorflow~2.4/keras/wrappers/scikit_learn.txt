# Module: tf.keras.wrappers.scikit_learn

Wrapper for using the Scikit-Learn API with Keras models.

## Classes

`class KerasClassifier`: Implementation of the scikit-learn classifier API for
Keras.

`class KerasRegressor`: Implementation of the scikit-learn regressor API for
Keras.
